# Changelog

Todos los cambios notables en este proyecto serán documentados en este archivo.

## [1.0.0-lite] - 2025-12-01

### Características iniciales
- Ordenamiento por columnas (ascendente/descendente)
- Filtrado por columnas (texto y select)
- Búsqueda global en toda la tabla
- Paginación completa con navegación
- Vista responsiva para móviles
- Soporte para español e inglés
- API pública para manipulación de datos
- Sin dependencias externas
